# Generador de Tablas de Multiplicar
Programa hecho en NodeJs para generar tablas de multiplicar

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -g, --generar  Se crea un archivo de la tabla de multiplicar
                                                      [boolean] [default: false]
  -b, --base     Base sobre la cual multiplicar              [number] [required]
  -h, --hasta    Hasta que numero se hara la multiplicacion
                                                          [number] [default: 10]
  -l, --listar   Listar tabla de multiplicar          [boolean] [default: false]
```
